# Внесение изменений в контрпики или текстовые описания чемпионов или контрпиков
Открытые данные о контрпиках и описаниях чемпионов **League of Legends**, используемые в LoL Challenge Tracker.
Любой игрок может предложить улучшения — ваши правки помогут сообществу видеть актуальную информацию. 

## Что внутри
- `champion_roles.json` — список чемпионов с ролями, краткими описаниями стиля игры и контрпиками (сильные/слабые стороны).

## Как предложить изменение
1. **Fork** этого репозитория.
2. Отредактируйте файл `champion_roles.json` в вашем форке.
3. Создайте Pull Request с описанием, что и почему вы изменили.
4. Автор проекта рассмотрит ваш PR, и если данные корректны, примет изменения.

## Правила редактирования
- Добавляйте или изменяйте только имена чемпионов в списках `strengthsIcons` и `weaknessesIcons`, если хотите изменить чемпионов в контрпиках.
- Если хотите изменить текстовые описания - изменяйте только текст в `playstyle` (описание чемпиона), `strengthsSummary` (против кого он силен) и `weaknessesSummary` (против кого он слаб)
- Используйте правильные английские ключи чемпионов (поле "id" из последней версии Data Dragon).
- Желательно указывать источник или обоснование для изменений (например, статистические данные или личный опыт).
- Не изменяйте структуру файла и не удаляйте другие поля.
- Проверьте, что JSON остаётся валидным после редактирования (можно использовать любой онлайн-валидатор).

Данные используются в приложении **LoL Challenge Tracker**
